# Atlas-helpdesk-project
Infrastructure helpdesk interne basée sur GLPI, déployée sur environnement virtualisé (ESXi/Proxmox) avec segmentation réseau multi-zones (DMZ / LAN / MGMT), reverse proxy Nginx/TLS, supervision Zabbix et Plan de Reprise d'Activité (PRA) documenté. Projet réalisé dans le cadre de l'appel d'offre pédagogique Helpdesk Atlas — Master 1 IRS.
